# Коментарии под постом

## Клиент - сервис чата

Когда пользователь заходит во вкладку "комментарии" под постом, он подписывается на канал <a href="https://github.com/deve-proj/chat/lib/chat/post.ex">post:<post_id></a>, и устанавливает с каналом web-socket соедиение, что позволяет ему видеть активность комментаторов в режиме реального времени.

## Сервис чата - новостной сервис

Когда сервис чата ( далее: чат ) получает сообщение от пользователя, он BROADCASTит это сообщение всем подписчикам канала И в фоновом режиме оповещает новостной сервис ( далее: новости ) о событии: новый комментарий ( Kafka ).
