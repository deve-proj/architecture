# Добавление комментария

При отправке POST запроса* на ручку /news/comment, делается запись в базе данных:

- Добавляем комментарий в коллекцию comments ( сваливаем всё в кучу )

- Обновляем <a href="../schema.json">массив комментариев</a> в посте с указанным при запросе post_id, добавляя в него Id только что записанного комментария ( Получается связь many:one )


*Рассматривается возможность <a href="../../chat/scenarios/comments-under-post.md">добавления Apache Kafka</a> для событий
